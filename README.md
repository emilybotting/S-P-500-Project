# S&P 500 Project

## Objective

This project analyses and visualises historical S&P 500 market data from 2014 to 2017 to explore price behaviour, trading volume dynamics, and volatility patterns. The goal is to identify trends, anomalies, and relationships that may support data-driven market insights.

## Technologies

* **Data Source:** Historical S&P 500 market data (CSV format)
* **Language:** DAX
* **Visualisation:** Power BI

## Data Architecture

* Data is extracted from CSV files
* Data is transformed and visualised in Power BI
* Analytical outputs are also visualised in Power BI
